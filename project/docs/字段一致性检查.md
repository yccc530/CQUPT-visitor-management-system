## 五、状态值一致性

| 状态类型 | 取值 |
|---|---|
| 预约状态 | `PENDING_HOST`、`HOST_CONFIRMED`、`PENDING_DEPT`、`HOST_REJECTED`、`APPROVED`、`REJECTED`、`CANCELED`、`REJECTED_BLACKLIST` |
| 访问状态 | `NOT_ENTERED`、`ENTERED`、`EXITED`、`OVERTIME`、`ABNORMAL` |
| 审批结果 | `APPROVED`、`REJECTED`、`PENDING` |
| 通行凭证状态 | `VALID`、`USED`、`EXPIRED`、`DISABLED` |
| 黑名单状态 | `ACTIVE`、`EXPIRED`、`REMOVED` |

## 六、检查结论

本阶段已将前端核心页面展示字段与后端返回字段对齐。数据库仍保持规范化设计，姓名、部门名、校门名等展示字段不冗余存储在业务表中，而是在后端返回前按外键补全。
