1. 本机运行依赖 MySQL、JDK 17、Maven、Node.js/npm、Python、XeLaTeX；换机器提交或答辩前需要提前检查环境。
2. `project/.tools/`、`backend/target/`、`frontend/dist/`、`node_modules/` 不建议提交，已由 `.gitignore` 控制。
3. 报告中已插入 PNG 设计图；如需进一步提升印刷质量，可在提交前将 PNG 替换为同名 PDF 矢量图。
4. 前端构建有大 chunk 警告，主要来自 ECharts 和 Element Plus，当前不影响演示。
5. 数据库默认账号为 `root / root`，如本地 MySQL 密码不同，请使用环境变量覆盖。
