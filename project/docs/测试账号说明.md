本阶段重新执行 `scripts/init_database.sh` 后，数据库中包含：`sys_user` 13 条、`visit_apply` 24 条、`access_record` 13 条、`blacklist` 4 条、`operation_log` 20 条，能够支撑前端核心页面和统计图表展示。

## 四、初始化方法

```bash
cd project
bash scripts/init_database.sh
```

脚本默认使用 `root / root` 连接本机 MySQL，可通过环境变量覆盖：`DB_HOST`、`DB_PORT`、`DB_USERNAME`、`DB_PASSWORD`。
