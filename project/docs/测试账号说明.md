## 四、重新初始化数据库

在 MySQL 客户端中按顺序执行以下脚本：

```bash
mysql -uroot -p < database/create_database.sql
mysql -uroot -p cqupt_visitor_system < database/schema.sql
mysql -uroot -p cqupt_visitor_system < database/seed.sql
```

如果使用项目后续脚本，可在生成 `scripts/init_database.sh` 后通过脚本统一初始化。
